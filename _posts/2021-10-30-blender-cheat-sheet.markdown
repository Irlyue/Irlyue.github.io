# Blender cheat sheet

## Common

`tab`: switch between edit and object mode



## Edit Mode

**selection**

`1`: point selection

`2`: edge selection

`3`: face selection



**extrude**

`E`: extrude

`Alt-E`: more extrude tools



**loop cut**

`CTRL-R`: loop cut, left click and than right click to center.
